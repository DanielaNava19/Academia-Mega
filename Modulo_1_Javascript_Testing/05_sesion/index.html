¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de programación que se utiliza para estructurar y mostrar páginas web. Es el lenguaje estándar para crear y diseñar sitios web.

¿Qué hace HTML?

  • Define la estructura de la página web
  • Organiza el contenido, como texto, imágenes, videos, botones y otros elementos de navegación
  • Indica a los navegadores cómo mostrar el contenido
  • Define la jerarquía de títulos, el flujo de párrafos y la ubicación de imágenes
